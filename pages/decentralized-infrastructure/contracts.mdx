# Moloch v3 (Baal) Overview

Moloch v3 is a minimal yet composable DAO template continuing work from the Moloch, Minion and Compound frameworks to make it easier for people to combine and command crypto assets with intuitive membership games. Moloch v3 is a governance layer that sits on top of a multisig treasury layer. It uses the Gnosis Zodiac standards to interface with the treasury. DAOhaus apps are built on top of the Moloch DAO framework.

[Read more about Moloch v3 (Baal) on the contract docs site.](https://baal-docs.vercel.app/)
