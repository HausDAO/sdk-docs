# [`dates.ts`](https://github.com/HausDAO/daohaus-monorepo/blob/develop/libs/common-utilities/src/utils/dates.ts)

Imports the `format` and `formatDistanceToNow` function from the `date-fns` library and defines functions to format dates and times in different ways.

## Functions

- `formatDateTimeFromSeconds`
- `formatShortDateTimeFromSeconds`
- `formatDateFromSeconds`
- `formatLongDateFromSeconds`
- `formatDistanceToNowFromSeconds`
