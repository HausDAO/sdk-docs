# [`posterTags.ts`](https://github.com/HausDAO/daohaus-monorepo/blob/develop/libs/common-utilities/src/constants/posterTags.ts)

Defines constants `POSTER_TAGS` and `TABULA_TAGS`.

`POSTER_TAGS` is an object with three properties - `summoner`, `daoProfileUpdate`, and `signalProposal`.

`TABULA_TAGS` is also an object with three properties - `PUBLICATION`, `PUB_ACTION`, and `ARTICLE_ACTION`.

`PosterTags` is a type that can be one of two values - `daohaus.summoner.daoProfile` or `daohaus.shares.daoProfile`.

## Types

- `PosterTags`

## Constants

- `POSTER_TAGS`
- `TABULA_TAGS`
