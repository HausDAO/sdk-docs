# [`contracts.ts`](https://github.com/HausDAO/daohaus-monorepo/blob/develop/libs/common-utilities/src/constants/contracts.ts)

This code defines a constant named `CONTRACTS`, which is an object that contains information about various Ethereum contracts. Each property of the `CONTRACTS` object is a map of contract addresses keyed by network ID. The code also defines a constant named `SHAMAN_PERMISSIONS`, which is an array of objects containing information about Shaman permissions.

## Types

- `ContractKey`
- `AddressKeyChain`
- `ContractAddressList`

## Constants

- `CONTRACTS: ContractAddressList`
- `SHAMAN_PERMISSIONS`
