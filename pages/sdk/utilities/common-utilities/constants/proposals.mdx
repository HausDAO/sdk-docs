# [`proposals.ts`](https://github.com/HausDAO/daohaus-monorepo/blob/develop/libs/common-utilities/src/constants/proposals.ts)

Defines a type for a proposal status, and creates a constant object that maps strings to proposal statuses.

## Types

- `ProposalStatus`

### Statuses

- Unsponsored
- Voting
- Grace
- Expired
- Cancelled
- Ready for Processing
- Failed
- Passed
- Processing Failed
- Unknown

## Constants

- `PROPOSAL_STATUS: { [index: string]: ProposalStatus }`
