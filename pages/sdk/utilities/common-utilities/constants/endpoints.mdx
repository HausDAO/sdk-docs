# [`endpoints.ts`](https://github.com/HausDAO/daohaus-monorepo/blob/develop/libs/common-utilities/src/constants/endpoints.ts)

It defines a constant called `ENDPOINTS` that is a `KeychainList`. `KeychainList` is a list of keys and values, where the keys are the name of the chains and the values are the URLs associated with those chains.

## Constants

- `ENDPOINTS: KeychainList`
